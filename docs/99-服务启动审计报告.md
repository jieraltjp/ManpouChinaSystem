## 2. 启动顺序

```
第 1 步：启动基础设施（可选）
  docker compose -f docker/compose.yaml up -d
  → Redis（6379）、MySQL（3306）、Kafka（9092）等

第 2 步：启动核心服务
  mvn spring-boot:run -DskipTests   # user-service（18081）必须先启动
  mvn spring-boot:run -DskipTests   # procurement-service（18083）

第 3 步：启动前端
  cd apps/web && npm run dev -- --host   # http://localhost:3000
```
