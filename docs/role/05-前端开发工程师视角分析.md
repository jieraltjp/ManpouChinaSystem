## 3. 核心代码模式

### 3.1 API 客户端（已实现）

```typescript
// api/client.ts — Axios 实例
const client = axios.create({
  baseURL: '/api/v1',
  timeout: 10_000,
})

// 请求拦截：注入 JWT
client.interceptors.request.use((config) => {
  const auth = useAuthStore()
  if (auth.token) {
    config.headers.Authorization = `Bearer ${auth.token}`
  }
  return config
})

// 响应拦截：统一错误处理
client.interceptors.response.use(
  (res) => res,
  (err) => {
    if (err.response?.status === 401) {
      useAuthStore().logout()
      window.location.href = '/login'
    }
    return Promise.reject(err)
  },
)
```

### 3.2 认证 Store（已实现）

```typescript
// stores/auth.ts
export const useAuthStore = defineStore('auth', () => {
  const token = ref<string | null>(null)
  const userInfo = ref<UserInfo | null>(null)

  const isAuthenticated = computed(() => !!token.value)

  async function login(cmd: LoginCmd) {
    const vo = await loginApi(cmd)
    token.value = vo.accessToken
    userInfo.value = decodeToken(vo.accessToken)
  }

  function logout() {
    token.value = null
    userInfo.value = null
  }

  return { token, userInfo, isAuthenticated, login, logout }
})
```

### 3.3 路由守卫（已实现）

```typescript
// router/index.ts
router.beforeEach((to, _from, next) => {
  const auth = useAuthStore()

  if (to.meta.requiresAuth !== false && !auth.isAuthenticated) {
    next({ name: 'Login', query: { redirect: to.fullPath } })
    return
  }

  if (to.name === 'Login' && auth.isAuthenticated) {
    next({ name: 'Dashboard' })
    return
  }

  next()
})
```

## 4. 组件开发规范

### 4.1 组件分层

| 层级 | 目录 | 说明 |
|------|------|------|
| 原子组件 | `components/atoms/` | Button, Input, Loading（无业务逻辑） |
| 分子组件 | `components/molecules/` | Form, Card（组合原子组件） |
| 有机体 | `components/organisms/` | Header, Sidebar（完整功能区块） |
| 页面 | `pages/` | 路由页面（编排组件） |

### 4.2 API 适配规范

```typescript
// ✅ 正确：从后端 Entity 转换为前端 ViewModel
export function toExamplePageItem(raw: ExampleDTO): ExamplePageItem {
  return {
    id: raw.id,
    name: raw.name,
    status: STATUS_MAP[raw.status] ?? raw.status,
    createTime: dayjs(raw.createTime).format('YYYY-MM-DD HH:mm'),
  }
}

// ❌ 错误：直接暴露后端字段名
// return { id: raw.id, name: raw.name, ...raw } // 违反防腐层
```

### 4.3 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 组件文件 | PascalCase | `ProcurementList.vue` |
| 组件名 | PascalCase | `ProcurementList` |
| Hook 函数 | camelCase，以 use 开头 | `useProcurement.ts` |
| Store | 小写下划线 | `procurement.ts` |
| API 函数 | camelCase，以 fetch/post/put/delete 开头 | `fetchProcurement` |
