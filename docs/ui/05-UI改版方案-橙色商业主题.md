# UI 改版方案：橙色商业主题焕新

> **定位**：将 Phase 0 前端从 Element Plus 默认蓝升级为专业商业橙色风格
> **覆盖页面**：登录页、布局框架、发注单管理页
> **原则**：不改结构，只改视觉层——颜色、字体、间距、阴影、光影

---

## 1. 当前问题分析

### 1.1 登录页
| 问题 | 说明 |
|------|------|
| 渐变背景无品牌感 | `#667eea → #764ba2` 是 Vue 官方示例色，与业务无关 |
| 卡片无层次 | 纯白卡片 + hover 阴影，缺乏质感 |
| 标题泛泛 | "MANPOU 企业管理系统" 无业务语义 |
| 无品牌色点缀 | 全程无橙色出现 |

### 1.2 布局框架
| 问题 | 说明 |
|------|------|
| 侧边栏深灰色太沉 | `#304156` 工业风，缺乏温暖感 |
| Logo 无图形 | 纯文字 "MANPOU"，不专业 |
| 顶栏无品牌色 | 纯白，缺乏统一感 |
| 菜单项无视觉层级 | 高亮色仍是默认蓝 |

### 1.3 发注单管理页
| 问题 | 说明 |
|------|------|
| 统计卡片各自为政 | 无视觉关联，缺乏数据驾驶舱感 |
| 状态 tag 色系混乱 | tag 类型直接用 Element 预设，不一致 |
| 操作按钮无层级 | primary/success/danger 平铺 |
| 表格无品牌色 | 表头、数据行、选中行均无橙色点缀 |

## 2. 色彩系统

### 2.1 调色板

```
主色系（橙色 — 品牌核心）
├── primary:        #E8650A   // 品牌橙（按钮、Logo、高亮）
├── primary-light:  #FF8C3A   // 浅橙（hover 态、徽标）
├── primary-dark:   #C4520A   // 深橙（active 态、按压）
├── primary-pale:   #FFF3EB   // 极浅橙（背景、标签底色）

中性色系
├── bg-page:        #F7F8FA   // 页面底色（浅灰白）
├── bg-card:        #FFFFFF   // 卡片底色
├── bg-sidebar:     #1E2533   // 侧边栏（深蓝灰，橙色对比）
├── bg-header:      #FFFFFF   // 顶栏底色
├── border:         #E8ECF1   // 边框线
├── text-primary:   #1A1A2E   // 主文字（近黑）
├── text-secondary: #6B7280   // 次要文字
├── text-muted:     #9CA3AF   // 占位/禁用文字

功能色
├── success:        #16A34A   // 成功（保留绿色语义）
├── warning:        #D97706   // 警告（橙色系）
├── danger:         #DC2626   // 危险（保留红色）
├── info:           #E8650A   // 信息（统一为橙色系）
```

### 5.2 状态标签：统一橙色语义

| 状态 | 原色 | 改版 |
|------|------|------|
