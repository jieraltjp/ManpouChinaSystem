## 2. 必须层（缺少则系统不可用）

> **判断标准**：去掉此项，至少一个核心业务流程断裂。

### 2.1 第1层 — 基础脚手架

| 组件 | 定位 | 状态 |
|------|------|------|
| JDK 21 | 运行时，ArchUnit 1.3.2 最低要求 | ✅ 已配置 |
| Spring Boot 3.2.5 | 框架基座 | ✅ 一致 |
| apps/java-service | Maven 父 POM，统一版本矩阵 | ✅ 已配置 |
| H2（内存） | 开发数据库，不依赖 Docker | ✅ 已配置 |
| apps/web | Vue 3 + Vite 前端 | ✅ 13000 端口运行中 |

### 2.2 第2层 — 核心业务（最小可运行集合）

| 组件 | 端口 | 定位 | 状态 |
|------|------|------|------|
| **manpou-allinone** | 18090 | 8 领域合一（procurement/logistics/qc/factory/product/warehouse/customs/finance/notification） | ✅ 运行中 |
| **user-service** | 18081 | JWT 签发、用户认证（保留独立） | ✅ 运行中 |
| **procurement-service** | 18083 | 🔴 仅骨架（procurement 已实装于 manpou-allinone 18090） | 🔴 废弃 |

```
用户登录 (18090) → JWT → 访问各领域接口 (18090)
前端 → api-gateway (18080) → manpou-allinone (18090)
```

**INTJ 判断**：manpou-allinone 用单体 jar 先行，8 领域合一，**后期按 Kafka Topic 边界逐步拆分**。

```

> 注意：procurement-service 骨架已废弃，真实发注单实现位于 manpou-allinone/procurement (18090)。

  ┌─────────────────────────────────────┐
  │ 1. api-gateway 启动验证 ✅           │
```

## 5. 决策矩阵（去/留判定）

```
         阻断核心开发？        影响线上稳定性？
             是       否          是       否
         ┌────────┬────────┐   ┌────────┬────────┐
    去   │  保留   │  保留   │   │  接入   │  暂缓   │
         ├────────┼────────┤   ├────────┼────────┤
    留   │  拆分   │  删除   │   │  立即修  │  观察   │
         └────────┴────────┘   └────────┴────────┘
```

**当前项目，逐项判定：**

| 组件 | 阻断核心开发？ | 影响线上稳定性？ | 决策 |
|------|:--------------:|:----------------:|------|
| manpou-allinone | ✅ 是 | ✅ 是 | ⚡ Phase 0 已完成，8 领域合一（procurement/logistics/qc/factory + 4 骨架模块） |
| user-service | ✅ 是 | — | 保留（第1优先） |
| procurement-service | ❌ 否 | — | procurement 业务已实现在 manpou-allinone；procurement-service 骨架可保留供未来独立拆分 |
| api-gateway | ✅ 是 | ✅ 是 | Phase 1 已接入 ✅ |
| Kafka 消费者 | ❌ 否 | ✅ 是 | Phase 3 接入 |
| product/warehouse/customs/logistics/finance | ❌ 否 | ❌ 否 | allinone 内实现，后期按 Topic 拆分 |
| notification-service | ❌ 否 | ❌ 否 | Phase 3 后 |
| Docker 基础设施 | ❌ 否 | ✅ 是 | Phase 2 启动 |
| CI/CD | ❌ 否 | ✅ 是 | Phase 4 激活 |
| 文档（role/pro/ui） | ❌ 否 | ❌ 否 | 保留（已存在） |
