## B. 契约一致性

### B1. 前端 → 后端 路由链路

```
前端请求                  Vite Proxy           后端Controller
─────────────────────────────────────────────────────────────
GET /api/v1/auth/login   →  localhost:18081  →  AuthController.login
GET /api/v1/auth/public-key →  localhost:18081 →  AuthController.publicKey
GET /api/v1/examples     →  localhost:18081  →  ExampleController.list
POST /api/v1/examples    →  localhost:18081  →  ExampleController.create
PUT /api/v1/examples/:id →  localhost:18081  →  ExampleController.update
DELETE /api/v1/examples/:id → localhost:18081 →  ExampleController.delete
```

### B2. API Gateway 路由 vs 实际后端

| Gateway 路径 | 路由目标 | 实际后端 Controller | 状态 |
|-------------|---------|-------------------|------|
| `/api/v1/auth/**` | 18081 | AuthController | ✅ |
| `/api/v1/products/**` | 18082 | ExampleController | ⚠️ products 无专属 API |
| `/api/v1/purchase-orders/**` | 18083 | ExampleController | ⚠️ procurement 无采购单 API |
| `/api/v1/warehouse/**` | 18084 | ExampleController | ⚠️ warehouse 无专属 API |
| `/api/v1/customs/**` | 18085 | ExampleController | ⚠️ customs 无专属 API |
| `/api/v1/logistics/**` | 18086 | ExampleController | ⚠️ logistics 无专属 API |
| `/api/v1/finance/**` | 18087 | ExampleController | ⚠️ finance 无专属 API |

**问题**：Gateway 配置了业务路径（`/purchase-orders/**`），但后端仅有占位符 ExampleController，无真实业务 API。
